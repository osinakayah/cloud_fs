## Requirement
php 7.1.x


##  How to install

1. Clone the repository
2. Rename .env.example to .env
3. Configure the database settings in the .env file
3. Run  this command ```php artisan jwt:secret```

4. Run ```php artisan migrate```

5. Run ```php artisan serve```
